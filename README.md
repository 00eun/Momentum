### 주요 구현 기능

- 실시간으로 현재 시간 표시
- 이름 새로고침시에도 브라우저에서 기억
- todo-list 작성, 삭제 가능 이름과 마찬가지로 새로고침 시 브라우저에서 기억
- 사용자의 현재 위치와 온도, 날씨 표시 (오른쪽 상단)
- 화면 하단에 랜덤 명언
- 랜덤으로 배경화면 표시
